Software"SUBLIME_TEXT

What is it? The SUBLIME_TEXT program is one of the most popular tools currently for both web developers and layout designers. It is free for sporadic use (and cheap if you want to use it professionally), lightweight, multiplatform, and has a plentiful catalog of plugins.

Functional Decomposition

Functional decomposition refers broadly to the process of solving a functional relation in its constituent parts, in such a way that the original function can be reconstructed (ie, recomposed) of the parts according to the composition.
In the project was applied to recognize the components and know about each of them, as well as its operation.
